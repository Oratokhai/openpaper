## Status: 🔵 Tier 2 — Build in Beta

---

## What It Is

A structured block that displays the same prompt sent to two or more models, showing their responses side by side for direct comparison.

---

## Why It Matters

Benchmarks and model comparisons are among the most-read AI content. Writers currently build these as manual tables or screenshots. A native comparison block makes this content richer, scannable, and credible — and no other platform has it.

---

## Features

| Feature | Description |
|---|---|
| Shared prompt header | Shows the prompt that was sent to all models |
| Side-by-side columns | Each model's response in its own column |
| Model attribution | Each column labelled with [[Model Mentions & Pages\|model mention]] |
| Date generated | When each response was captured |
| Highlight / annotate | Writer can highlight specific differences |
| Mobile stacking | Columns stack vertically on small screens |
